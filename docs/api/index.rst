.. _sec-api:

#####
 API
#####

The core of **xNVMe** is a library implemented in the C programming language.
The C library implements the foundational C API, to other languages bind to via
FFIs. Thus, this section will document the C API, and provide usage examples.
Additionally, then language bindings to Python and Rust are described.
